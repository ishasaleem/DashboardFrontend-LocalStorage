**Project Objective**

professional Admin Dashboard & Product Management Web Application using modern frontend technologies.
The application  store and manage data using Browser Local Storage 

 **Technologies used**

React.js 

Bootstrap OR Tailwind CSS

HTML5 & CSS3

Browser Local Storage

Git & GitHub

**Project Features**
1. Admin Dashboard

2. Product Management Screen

3. Add/Edit Product Screen


**Installation & Usage**

Clone the repository:

git clone https://github.com/your-username/your-repo.git

Navigate to project folder:

cd your-repo

Install dependencies:

npm install

Run the project:

npm start

Open http://localhost:3000
 in your browser.
